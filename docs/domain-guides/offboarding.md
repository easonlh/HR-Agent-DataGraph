# 离职分类体系领域指南

## 概述

离职分类体系是 HR-Agent-DataGraph 最核心的数据域之一。它将中国企业的离职场景标准化为可机器处理的分类编码，同时保留完整的法律引用和业务上下文。

## 法律基础

### 《劳动合同法》核心条款

| 条款 | 内容 | 分类映射 |
|------|------|---------|
| 第36条 | 协商解除 | `MUTUAL_AGREEMENT` |
| 第37条 | 劳动者提前通知解除 | `VOLUNTARY.RESIGNATION_NORMAL` |
| 第39条 | 过失性辞退 | `INVOLUNTARY.TERMINATION_WITH_CAUSE` |
| 第40条 | 非过失性辞退 | `INVOLUNTARY.TERMINATION_WITHOUT_CAUSE` |
| 第41条 | 经济性裁员 | `INVOLUNTARY.LAYOFF` |
| 第44条 | 合同终止 | `CONTRACT_EXPIRY` |
| 第46条 | 经济补偿适用情形 | `severanceApplicable: true` |
| 第47条 | 经济补偿计算标准 | `severanceFormula: "N"` |
| 第87条 | 违法解除赔偿 | `severanceFormula: "2N"` |

## 经济补偿金公式

### N（标准经济补偿）

```
N = 工作年限 × 月平均工资
```

- 工作年限：≥6个月按1年算，<6个月按半个月算
- 月平均工资：离职前12个月的平均工资
- 上限：当地上年度职工月平均工资的3倍，且最多补偿12年

### N+1（代通知金）

适用于未提前30天通知的非过失性辞退：

```
N+1 = N + 上个月工资
```

### 2N（违法解除赔偿金）

适用于违法解除劳动合同：

```
2N = N × 2
```

## 退出流程

标准离职流程包含10个步骤，法定15个工作日内完成（劳动合同法第50条）：

1. 提交离职申请
2. 直属上级审批
3. HR审核
4. 工作交接
5. 资产归还
6. 权限回收
7. 薪资结算
8. 社保公积金转移
9. 开具离职证明
10. 档案归档
