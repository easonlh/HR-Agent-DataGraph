# 行政区划数据

## 标准依据

本数据基于 **GB/T 2260《中华人民共和国行政区划代码》** 编制。

- 省级代码：2位（如 `11` = 北京市）
- 地级代码：4位（如 `1101` = 北京市市辖区）
- 县级代码：6位（如 `110101` = 东城区）

## 数据层级

| 层级 | 字段数 | 说明 |
|------|--------|------|
| `provinces.json` | 34 | 省、自治区、直辖市、特别行政区 |
| `cities.json` | 样本 | 地级市、自治州、地区、盟 |

## 用途

- 员工籍贯、户籍地址、社保缴纳地的标准化
- 跨区域企业的组织架构与薪酬地域差异分析
- 与社保公积金系统的行政区划编码对接
