# 高校与学历数据

## 分类体系

中国高等教育具有独特的层级分类：

| 分类 | 说明 |
|------|------|
| 985工程 | 39所世界一流大学建设高校 |
| 211工程 | 112所重点建设高校 |
| 双一流 | 世界一流大学和一流学科建设高校（2017年起替代985/211） |

## 学历 vs 学位

- **学历（Diploma/Credential）**：学习经历的证明，如本科毕业证书
- **学位（Academic Degree）**：学术水平的认证，如学士学位证书

两者是独立概念：有学历不一定有学位（结业），有学位必须有对应的学历。

## 数据文件

| 文件 | 说明 |
|------|------|
| `institutions.json` | 高校样本数据（985/211重点高校） |
| `degree-types.json` | 学历学位层级完整定义 |

## 用途

- 学历背景的标准化校验
- 招聘筛选中的院校层级评估
- OCR学历证书识别结果的标准化映射
