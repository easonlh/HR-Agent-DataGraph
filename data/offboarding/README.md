# 离职分类体系

## 法律依据

本分类体系基于以下法律法规：

- **《中华人民共和国劳动合同法》**（2012修正）— 第36-50条
- **《中华人民共和国劳动合同法实施条例》** — 第18-24条
- **《中华人民共和国劳动法》** — 第24-32条

## 分类概览

| 类型 | 法律条款 | 经济补偿 |
|------|---------|---------|
| 主动离职 | 第37条 | 无 |
| 过失性辞退 | 第39条 | 无 |
| 非过失性辞退 | 第40条 | N |
| 经济性裁员 | 第41条 | N |
| 协商解除 | 第36条 | 协商（≥N） |
| 合同到期 | 第44条 | N（特殊情况） |
| 退休 | — | 无 |

## 经济补偿金公式

- **N** = 工作年限 × 月平均工资（≥6个月按1年算，<6个月按半个月算）
- **N+1** = N + 1个月代通知金（未提前30天通知时）
- **2N** = 违法解除的赔偿金
