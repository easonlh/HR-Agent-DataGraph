# AI Agent 技能图谱

## 设计理念

AI Agent 技能图谱是 HR 数据域与 AI 自动化之间的桥梁。它定义了：

1. **技能分类体系** — 多维度、可加权的能力评估框架
2. **证据类型** — AI Agent 可采集的行为数据源定义
3. **推理规则** — 从原始数据到技能信号的映射逻辑
4. **能力向量** — 标准化的员工能力快照格式

## 数据流水线

```
数据采集 (INGEST)
    │  Git提交、会议记录、代码评审、OCR证件
    ▼
特征提取 (EXTRACT)
    │  应用推理规则，提取技能信号
    ▼
信号聚合 (AGGREGATE)
    │  多源信号加权聚合，计算置信度
    ▼
结果输出 (OUTPUT)
    生成标准化能力向量
```

## 数据文件

| 文件 | 说明 |
|------|------|
| `skill-graph.json` | 技能分类、证据类型、推理规则、能力向量规范 |
| `document-parsing-spec.json` | OCR证件解析输入规范（身份证/学历证/离职证明） |

## 用途

- 人才盘点与继任规划的AI辅助分析
- 基于行为数据的能力自动评估
- 招聘场景中候选人能力向量匹配
