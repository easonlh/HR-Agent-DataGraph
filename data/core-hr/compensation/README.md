# 薪酬福利数据

## 中国薪酬结构

中国企业薪酬通常由以下部分组成：

```
┌─────────────────────────────────────┐
│            应发工资总额              │
├──────────────┬──────────────────────┤
│   固定薪酬    │      浮动薪酬        │
│  ├ 基本工资   │  ├ 绩效奖金          │
│  ├ 岗位津贴   │  ├ 年终奖            │
│  └ 工龄补贴   │  └ 项目奖金          │
├──────────────┴──────────────────────┤
│           法定扣除                   │
│  ├ 养老保险（个人）                  │
│  ├ 医疗保险（个人）                  │
│  ├ 失业保险（个人）                  │
│  ├ 住房公积金（个人）                │
│  └ 个人所得税                        │
├─────────────────────────────────────┤
│           企业法定支出               │
│  ├ 养老保险（企业）                  │
│  ├ 医疗保险（企业）                  │
│  ├ 失业/工伤/生育保险（企业）         │
│  └ 住房公积金（企业）                │
└─────────────────────────────────────┘
```

## 数据文件

| 文件 | 说明 |
|------|------|
| `components.json` | 薪酬组件分类（固定/浮动/津贴/福利） |
| `statutory-components.json` | 法定福利结构（五险一金+个税） |

## 用途

- 薪酬方案设计的标准化组件库
- 薪资计算引擎的字段定义参考
- 跨企业薪酬数据对标分析
